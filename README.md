# Go Adventures

Building a Project in Marroco

# Commands

```
-- Create App

npm init -y

npm i -D gh-pages

-- Install Sass

npm i -D sass
npm uninstall sass

npm install -D sass@1.69.5

npx sass assets/scss/main.scss assets/css/main.css

-- Run

php -S localhost:8080
http://localhost:8080/

```
